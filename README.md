# Not0Day — Cybersecurity Portfolio

> **Handle:** Not0Day  
> **Site:** [notzeroday.me](https://notzeroday.me)  
> **Aesthetic:** Stormtrooper — Pure Black, Pure White, Neon Cyan

A professional, high-end cybersecurity portfolio built with [Astro](https://astro.build/).
Showcasing penetration testing projects, CTF writeups, security research, and more.

## Stack

- **Framework:** [Astro](https://astro.build/) v4
- **Styling:** Scoped CSS (no external CSS framework)
- **Hosting:** GitHub Pages → `notzeroday.me`
- **Theme:** Stormtrooper aesthetic (`#080c10` / `#ffffff` / `#00ffff`)

## Development

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # production build → dist/
npm run preview  # preview production build
```
